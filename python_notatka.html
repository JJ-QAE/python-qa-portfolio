Podstawy
print
Wydrukuj coś
Wyświetla tekst lub zmienną w konsoli.
print("Hello, world!")
print("Mam na imię " + name)
zmienna
Przechowaj wartość
Zmienna to pudełko w którym trzymasz dane.
name = "Jowita"
age = 21
print("Cześć, " + name)
input
Zapytaj użytkownika
Zapisz odpowiedź w zmiennej żeby móc jej użyć.
name = input("Jak masz na imię? ")
print("Cześć, " + name)
int()
Zamień tekst na liczbę
input() zawsze zwraca tekst — użyj int() jeśli potrzebujesz liczby.
age = int(input("Ile masz lat? "))
print(age + 10)
float()
Zamień tekst na liczbę z przecinkiem
Jak int() ale dla liczb z przecinkiem (np. wzrost, waga).
wzrost = float(input("Wzrost (m)? "))
print(f"{wzrost:.1f}") # :.1f = 1 miejsce po przecinku
Stringi (tekst)
indeksowanie
Pobierz znak ze stringa
Liczymy od 0. -1 to zawsze ostatni znak.
word = "Python"
print(word[0])   # P
print(word[-1])  # n (ostatni)
print(len(word)) # 6 (długość)
łączenie
Łącz stringi
Użyj + żeby skleić teksty razem.
first = "Hej, "
second = "Jowita!"
print(first + second)
# Hej, Jowita!
Pętle
for loop
Powtórz coś X razy
Wcięcie mówi Pythonowi co należy do pętli.
for i in range(1, 11):
    print(i)  # drukuje 1-10
range()
Generuj liczby
Trzy argumenty: start, stop (nie include!), krok.
range(1, 11)     # 1 do 10
range(0, 10)     # 0 do 9
range(2, 21, 2)  # parzyste: 2,4,6...
while loop
Powtarzaj dopóki warunek prawdziwy
Używaj gdy nie wiesz ile razy pętla ma się wykonać.
while guess != answer:
    print("Spróbuj jeszcze raz!")
    guess = input("Odpowiedź: ")
print("Dobrze!")
for po liście
Przejdź przez każdy element
Prostszy sposób niż range() gdy masz listę.
fruits = ["jabłko", "banan"]
for item in fruits:
    print(item)
Funkcje
def
Stwórz własną funkcję
Napisz raz, używaj ile razy chcesz.
def greet(name):
    print("Cześć, " + name + "!")

greet("Jowita")
return
Zwróć wartość z funkcji
return pozwala użyć wyniku funkcji dalej w kodzie.
def oblicz_bmi(waga, wzrost):
    return waga / (wzrost * wzrost)

bmi = oblicz_bmi(79, 1.67)
print(f"BMI: {bmi:.1f}")
if/elif/else
Warunek
== sprawdza czy dwie rzeczy są równe.
if bmi < 18.5:
    print("niedowaga")
elif bmi <= 24.99:
    print("prawidłowa")
else:
    print("nadwaga")
Listy i Słowniki
lista
Zbiór elementów
Lista trzyma wiele rzeczy naraz w jednej zmiennej.
fruits = ["jabłko", "banan", "wiśnia"]
print(fruits[0])   # jabłko
fruits.append("mango")
print(len(fruits))  # 4
słownik
Klucz: wartość
Słownik przechowuje pary klucz-wartość.
person = {"name": "Jowita", "age": 21}
print(person["name"])  # Jowita
person["city"] = "Warsaw"
F-stringi
f-string
Wstaw zmienną do tekstu
Zamiast łączyć + możesz wstawić zmienną prosto w tekst używając {}.
name = "Jowita"
score = 3
print(f"Cześć, {name}!")
print(f"Wynik: {score}/5")

# w URL-ach:
post_id = 1
url = f"https://api.com/posts/{post_id}"
Moduły
random
Losuj elementy
random.choice() losuje jeden element z listy.
import random
print(random.choice("abcdef"))
# wylosuje jedną literę
string
Gotowe zestawy znaków
Nie musisz pisać wszystkich liter ręcznie.
import string
string.ascii_letters  # a-z A-Z
string.digits         # 0-9
string.punctuation    # !@#$%...
datetime
Dzisiejsza data
Moduł datetime daje dostęp do daty i czasu.
from datetime import date
dzisiaj = date.today()
print(dzisiaj)  # 2026-06-05
Pliki i JSON
zapis "w"
Zapisz do pliku
"w" = write. Nadpisuje plik.
with open("plik.txt", "w") as plik:
    plik.write("Cześć!")
odczyt "r"
Odczytaj plik
"r" = read. Odczytuje zawartość pliku.
with open("plik.txt", "r") as plik:
    zawartosc = plik.read()
print(zawartosc)
dopisz "a"
Dopisz do pliku
"a" = append. Dopisuje na końcu — zachowuje co było.
with open("plik.txt", "a") as plik:
    plik.write(y + "\n")
# \n = nowa linia
json.dump / json.load
Zapisz i odczytaj słownik
JSON zamienia słownik na tekst i z powrotem.
import json
# zapis:
with open("plik.json", "w") as plik:
    json.dump(slownik, plik)
# odczyt:
with open("plik.json", "r") as plik:
    slownik = json.load(plik)
Obsługa błędów
try/except
Złap błąd
Zamiast się wysypać — zareaguj normalnie.
try:
    waga = float(input("Ile ważysz? "))
except ValueError:
    print("To nie jest liczba!")
while + break
Pytaj dopóki poprawne
while True = pętla nieskończona, break = wyjdź gdy się uda.
while True:
    try:
        waga = float(input("Ile ważysz? "))
        break  # wyjdź gdy ok!
    except ValueError:
        print("Spróbuj jeszcze raz!")
Pamiętaj: = to przypisanie (name = "Jowita"), a == to porównanie (if name == "Jowita"). Wcięcie (spacja przed kodem) mówi Pythonowi co należy do pętli lub funkcji.
